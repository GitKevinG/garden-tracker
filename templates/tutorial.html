Getting Started Guide

Follow this seed-to-harvest workflow to get the most out of Garden Tracker.

1. Add Seeds

Start by adding your seed varieties to the inventory. Include days to maturity and weeks to transplant so the app can calculate planting schedules and expected harvest dates.

Add Seeds

2. Start Seedlings

Select a seed variety and set a sow date to start a seedling batch. The seed inventory automatically decrements. Track germination dates and viable counts as they grow.

Start Seedlings

3. Pot Up Seedlings

When seedlings are 14+ days old, pot them up to larger intermediate containers before transplanting outdoors. Record the pot size and quantity potted up.

View Seedlings

4. Set Up Containers

Add your grow bags with size and location info. The app calculates plant capacity based on bag size and plant size category. You can also set up hydroponic systems with bags and emitters.

Add Grow Bag Add Hydro System

5. Transplant

Create a plant from a ready seedling and assign it to a container. The seedling count updates automatically, and expected harvest dates are calculated from the seed's maturity info.

Transplant

6. Track Progress

Log observations on each plant: height, growth stage, health rating, issues, and actions taken. Use progress logs to spot trends and catch problems early.

View Plants

7. Record Harvests

Log each harvest with weight or count, unit, and a quality rating. The plant's status updates to "producing" after the first harvest, and yield totals are tracked automatically.

Record Harvest

8. Garden Layout

Arrange your grow bags and hydro systems on a visual top-down grid. Containers are grouped by location into zones. On desktop, drag and drop to place; double-click a placed container to remove it. On mobile, tap to select then tap a grid cell to place; long-press to remove. Positions are saved automatically.

Open Garden Layout

9. Seed Planner

Use the Seed Planner to create bag allocation plans. Select varieties, set bag counts, and the calculator figures out how many seeds to start based on germination rates and capacity.

Open Seed Planner

10. Calendar & Analytics

View the planting calendar to see seed start dates, transplant windows, and expected harvests based on your frost dates. Use analytics to compare variety performance across seasons.

View Calendar View Analytics
